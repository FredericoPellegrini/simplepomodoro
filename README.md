# simplepomodoro
O Simple Pomodoro é um cronômetro online minimalista focado em produtividade. Sem distrações, ele intercala automaticamente ciclos de 25 minutos de foco com 5 minutos de pausa. Possui alerta sonoro suave, barra de progresso visual e atualiza o tempo direto na aba do navegador. Feito de forma simples e direta com HTML, CSS e JavaScript puro.
